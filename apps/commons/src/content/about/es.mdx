# Acerca de

LEAF-Writer-Commons forma parte del conjunto de herramientas [LEAF Commons](https://www.leaf-vre.org/docs/about-leaf/leaf-commons), un conjunto de herramientas web, modulares y fáciles de usar que permiten la codificación de texto, el reconocimiento de entidades con nombre, la anotación web, el análisis y la publicación de texto sin necesidad de aprender lenguajes de marcado/codificación, y facilitan la movilidad entre herramientas interoperables. Este conjunto de herramientas, disponible gratuitamente, facilita los flujos de trabajo académicos digitales para la producción y publicación colaborativa de ediciones académicas y documentales, y colecciones en la web, sin necesidad de instalar software. El conjunto LEAF Commons permite el uso de herramientas individuales para fines específicos o un flujo de trabajo integral, desde la transcripción hasta la publicación, lo que permite una amplia gama de usos pedagógicos y de investigación.

[LEAF](https://www.leaf-vre.org/) significa Marco Académico de Edición Vinculada (Linked Editing Academic Framework), un conjunto de software colaborativo que proporciona un conjunto de herramientas modulares para la edición y publicación de texto. Las herramientas de LEAF Commons constituyen una infraestructura accesible, sencilla y gratuita para que los investigadores produzcan textos, ediciones o colecciones en línea, ya sea para la docencia o para la investigación y la colaboración sostenibles. Las herramientas de LEAF Commons están disponibles gratuitamente en el navegador, lo que facilita la colaboración y la publicación a través de Github, además de permitir el almacenamiento local o la instalación en una pila local. Las herramientas de LEAF Commons promueven la reutilización de metodologías de datos, de acuerdo con los principios de datos [FAIR](https://www.go-fair.org/fair-principles/) (Encontrables, Accesibles, Interoperables y Reutilizables). LEAF utiliza software de código abierto, plataformas de acceso abierto y estándares internacionales abiertos para las mejores prácticas en codificación de texto ([TEI-XML](https://tei-c.org)) y anotación web ([RDF](https://www.w3.org/RDF/)).

## LEAF-Writer

LEAF-Writer es un editor multilingüe de Lenguaje de Marcado Extensible (XML) de código abierto y acceso abierto que se ejecuta en un navegador web y ofrece a académicos y estudiantes una experiencia de edición de texto enriquecida sin necesidad de descargar, instalar ni configurar software propietario, pagar cuotas de suscripción ni aprender lenguajes de programación complejos. Este entorno de edición intuitivo incorpora los estándares de las Iniciativas de Codificación de Texto (TEI) y el Marco de Descripción de Recursos (RDF). Incorpora la conciliación con identificadores de datos abiertos vinculados y autoridades específicas del proyecto como parte del proceso de codificación. LEAF-Writer es especialmente valioso en contextos pedagógicos, ya que permite a los docentes enseñar a los estudiantes buenas prácticas de codificación sin necesidad de enseñarles a codificar ni instalar software localmente.

### Características principales

- Edición de documentos con reconocimiento de esquemas, incluyendo validación con esquemas accesibles desde la web
- Compatibilidad con hojas de estilo en cascada (CSS) para proporcionar una vista WYSIWYG de los documentos, así como una vista con etiquetas
- Capacidad para extraer referencias a entidades con nombre (personas, lugares u organizaciones) de referencias XML ya etiquetadas dentro de un documento para generar anotaciones web
- Capacidad para buscar y seleccionar identificadores para etiquetas de entidades con nombre (personas, organizaciones, lugares o títulos) de las siguientes autoridades de Datos Abiertos Vinculados: [DBPedia](https://dbpedia.org/), [Geonames](https://www.geonames.org/), [Getty](https://www.getty.edu/), [GND](https://www.dnb.de/EN/Professionell/Standardisierung/GND/gnd_node.html), [Proyecto LINCS](https://lincsproject.ca), [VIAF](https://www.viaf.org/) y [Wikidata](https://www.wikidata.org/), o archivos de autoridad específicos del proyecto.
- Generación de anotaciones de Datos Vinculados correspondientes a entidades con nombre y anotaciones de documentos (fechas, notas, citas, correcciones, enlaces, palabras clave) recién etiquetadas en XML-RDF o JSON-LD, conforme al Modelo de Datos de Anotación Web.
- Validación XML continua.
- Opciones de etiquetado con restricciones Shema.

### Compatibilidad con TEI

LEAF-Writer está diseñado para funcionar con las personalizaciones del esquema de la Iniciativa de Codificación de Texto (TEI) (https://tei-c.org/guidelines/customization/#section-1) proporcionadas por el Consorcio TEI. De fábrica, LEAF-Writer admite los siguientes esquemas: [TEI All](https://www.tei-c.org/release/xml/tei/custom/schema/relaxng/tei_all.rng), [TEI LITE](https://www.tei-c.org/release/xml/tei/custom/schema/relaxng/tei_lite.rng), [TEI Simple Print](https://www.tei-c.org/release/xml/tei/custom/schema/relaxng/tei_simplePrint.rng), [jTEI Article](https://www.tei-c.org/release/xml/tei/custom/schema/relaxng/tei_jtei.rng) y [Orlando](https://cwrc.ca/schemas/orlando_entry.rng).

### Esquemas Personalizados

LEAF-Writer también puede funcionar con esquemas personalizados. Al abrir un documento, LEAF-Writer comprueba el elemento raíz y la definición del esquema. Actualmente, LEAF-Writer admite tres elementos raíz diferentes: `TEI`, `ORLANDO` y `CWRCENTRY`. Si el elemento raíz no es compatible con el esquema, puede agregarlo como un esquema personalizado personal. LEAF-Writer guarda la información del esquema en el almacenamiento local del navegador. El esquema estará disponible localmente mientras mantenga la sesión iniciada.

## Almacenamiento de Documentos

LEAF-Writer Commons utiliza GitHub para el almacenamiento, control de versiones y compartición de documentos. Para aprovechar estas funciones, debe iniciar sesión en una cuenta de GitHub. Además, puede abrir documentos pegando un XML o subiendo un archivo desde su ordenador. También puede descargar el archivo directamente a su dispositivo. Opcionalmente, puede usar LEAF-Writer sin una cuenta externa. En este caso, solo podrá cargar y guardar archivos desde su computadora.

Puede usar LEAF-Writer para editar documentos XML o crear nuevos documentos a partir de plantillas. Aquí encontrará plantillas y documentos de ejemplo para comenzar.
