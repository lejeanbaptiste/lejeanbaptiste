# Despre

LEAF-Writer-Commons face parte din [suita de instrumente LEAF Commons](https://www.leaf-vre.org/docs/about-leaf/leaf-commons), un set de instrumente modulare, ușor de utilizat, bazate pe web, care acceptă codificarea textului, recunoașterea entităților denumite, adnotarea web, analiza textului și publicarea fără ca utilizatorii să fie nevoiți să învețe limbaje de marcare/codare și facilitează trecerea între instrumente interoperabile. Această suită de instrumente disponibilă gratuit acceptă fluxuri de lucru academice digitale pentru producția și publicarea colaborativă a edițiilor și colecțiilor academice și documentare pe web, fără a fi nevoie de instalarea de software. Suita LEAF Commons permite utilizarea instrumentelor individuale în scopuri specifice sau acceptă un flux de lucru complet, de la transcriere la publicare, permițându-i să deservească o gamă largă de utilizări în cercetare și pedagogie.

[LEAF](https://www.leaf-vre.org/) reprezintă Linked Editing Academic Framework, o suită de software colaborativă care oferă un set de instrumente modulare pentru editarea și publicarea de texte. Instrumentele LEAF Commons constituie o infrastructură accesibilă, fără bariere și fără costuri pentru cercetători, care permite producerea de texte, ediții sau colecții online, fie în scopuri didactice, fie pentru cercetare și colaborare în mod sustenabil. Instrumentele LEAF Commons sunt disponibile gratuit în browser, permițând colaborarea și publicarea prin Github, pe lângă stocarea locală sau instalarea într-un stack local. Instrumentele LEAF Commons promovează reutilizarea metodologiilor de date în conformitate cu principiile datelor [FAIR](https://www.go-fair.org/fair-principles/) (Findă, Accesibilă, Interoperabilă și Reutilizabilă). LEAF utilizează software open-source, platforme cu acces liber și standarde internaționale deschise pentru cele mai bune practici în codificarea textului ([TEI-XML](https://tei-c.org)) și adnotarea web ([RDF](https://www.w3.org/RDF/)).

## LEAF-Writer

LEAF-Writer este un editor XML (Extensible Markup Language) multilingv, open-source și cu acces liber, care rulează într-un browser web și oferă elevilor și studenților lor o experiență bogată de editare textuală, fără a fi nevoie să descarce, să instaleze și să configureze software proprietar, să plătească taxe de abonament permanente sau să învețe limbaje de codare complexe. Acest mediu de editare ușor de utilizat încorporează standardele Text Encoding Initiatives (TEI) și Resource Description Framework (RDF). Acesta încorporează reconcilierea cu identificatori de date deschise legați și autorități specifice proiectului ca parte a procesului de codare. LEAF-Writer este deosebit de valoros în contexte pedagogice, permițând instructorilor să îi învețe pe studenți bune practici de codare fără a fi nevoie să îi învețe și să programeze sau să instaleze software local.

### Caracteristici principale

- Editare a documentelor bazată pe scheme, inclusiv validare, în raport cu scheme accesibile web
- Suport pentru foi de stil în cascadă (CSS) pentru a oferi o vizualizare WYSIWYG a documentelor, precum și o vizualizare care arată etichetele
- Capacitatea de a extrage referințe la entități denumite (persoane, locuri sau organizații) din referințe XML deja etichetate dintr-un document pentru a genera adnotări web
- Capacitatea de a căuta și selecta identificatori pentru etichetele entităților denumite (persoane, organizații, locuri sau titluri) din următoarele autorități Linked Open Data: [DBPedia](https://dbpedia.org/), [Geonames](https://www.geonames.org/), [Getty](https://www.getty.edu/), [GND](https://www.dnb.de/EN/Professionell/Standardisierung/GND/gnd_node.html), [LINCS Project](https://lincsproject.ca), [VIAF](https://www.viaf.org/) și [Wikidata](https://www.wikidata.org/) sau fișiere de autoritate specifice proiectului.
- Generarea de adnotări Linked Data corespunzătoare entităților denumite recent etichetate și adnotărilor documentelor (date, note, citări, corecții, linkuri, cuvinte cheie) în XML-RDF sau JSON-LD, conforme cu Modelul de date pentru adnotări web
- Validare XML continuă
- Opțiuni de etichetare constrânse de shema

### Suport TEI

LEAF-Writer este conceput să funcționeze cu [personalizări ale schemei Text Encoding Initiative (TEI)](https://tei-c.org/guidelines/customization/#section-1) furnizate de Consorțiul TEI. LEAF-Writer acceptă următoarele scheme implicite: [TEI All](https://www.tei-c.org/release/xml/tei/custom/schema/relaxng/tei_all.rng), [TEI LITE](https://www.tei-c.org/release/xml/tei/custom/schema/relaxng/tei_lite.rng), [TEI Simple Print](https://www.tei-c.org/release/xml/tei/custom/schema/relaxng/tei_simplePrint.rng), [jTEI Article](https://www.tei-c.org/release/xml/tei/custom/schema/relaxng/tei_jtei.rng) și [Orlando](https://cwrc.ca/schemas/orlando_entry.rng).

### Scheme personalizate

LEAF-Writer poate funcționa și cu scheme personalizate. Când deschideți un document, LEAF-Writer verifică elementul rădăcină și definiția schemei. În prezent, LEAF-Writer acceptă trei elemente rădăcină diferite: `TEI`, `ORLANDO` și `CWRCENTRY`. Dacă elementul rădăcină este acceptat de schemă nu, îl puteți adăuga ca o schemă personalizată personală. LEAF-Writer salvează informațiile schemei în memoria locală a browserului. Apoi, schema va fi disponibilă local atâta timp cât rămâneți conectat.

## Stocare documente

LEAF-Writer Commons utilizează GitHub pentru stocarea, versionarea și partajarea documentelor. Pentru a profita de aceste funcții, trebuie să fiți conectat la un cont GitHub. În plus, puteți deschide documente lipind un XML sau încărcând un fișier de pe computer. De asemenea, puteți descărca fișierul direct pe dispozitiv. Opțional, puteți utiliza LEAF-Writer fără niciun cont extern, caz în care veți putea încărca și salva documente doar de pe computer.

Puteți utiliza LEAF-Writer pentru a edita documente XML sau pentru a produce documente noi din șabloane. Există șabloane și documente exemplu aici pentru a începe.
