# Sobre

O LEAF-Writer-Commons faz parte do [pacote de ferramentas LEAF Commons](https://www.leaf-vre.org/docs/about-leaf/leaf-commons), um conjunto de ferramentas modulares, fáceis de usar e baseadas na web, que oferece suporte à codificação de texto, reconhecimento de entidades nomeadas, anotação na web, análise de texto e publicação, sem que os usuários precisem aprender linguagens de marcação/codificação, além de facilitar a migração entre ferramentas interoperáveis. Este pacote de ferramentas, disponível gratuitamente, oferece suporte a fluxos de trabalho acadêmicos digitais para a produção e publicação colaborativa de edições acadêmicas e documentais, bem como coleções na web, sem a necessidade de instalação de software. O pacote LEAF Commons permite o uso de ferramentas individuais para fins específicos ou oferece suporte a um fluxo de trabalho completo, da transcrição à publicação, permitindo que atenda a uma ampla gama de usos em pesquisa e pedagogia.

[LEAF](https://www.leaf-vre.org/) significa Linked Editing Academic Framework (Estrutura Acadêmica de Edição Ligada), um pacote de software colaborativo que fornece um conjunto de ferramentas modulares para edição e publicação de texto. As ferramentas LEAF Commons constituem uma infraestrutura acessível, de fácil utilização e gratuita para que pesquisadores produzam textos, edições ou coleções online, seja para fins de ensino ou para a realização de pesquisas e colaborações de forma sustentável. As ferramentas LEAF Commons estão disponíveis gratuitamente no navegador, permitindo a colaboração e publicação via GitHub, além de possibilitar o armazenamento local ou a instalação em um ambiente local. As ferramentas LEAF Commons promovem a reutilização de metodologias de dados em consonância com os princípios [FAIR](https://www.go-fair.org/fair-principles/) (Localizáveis, Acessíveis, Interoperáveis ​​e Reutilizáveis). O LEAF utiliza software de código aberto, plataformas de acesso aberto e padrões internacionais abertos para as melhores práticas em codificação de texto ([TEI-XML](https://tei-c.org)) e anotação web ([RDF](https://www.w3.org/RDF/)).

## LEAF-Writer

O LEAF-Writer é um editor de XML (Extensible Markup Language) multilíngue, de código aberto e acesso aberto, que funciona em um navegador web e oferece a acadêmicos e seus alunos uma rica experiência de edição de texto sem a necessidade de baixar, instalar e configurar softwares proprietários, pagar taxas de assinatura contínuas ou aprender linguagens de programação complexas. Este ambiente de edição amigável incorpora os padrões da Text Encoding Initiative (TEI) e do Resource Description Framework (RDF). Ele inclui a reconciliação com identificadores de dados abertos vinculados e autoridades específicas do projeto como parte do processo de codificação. O LEAF-Writer é particularmente valioso em contextos pedagógicos, permitindo que os instrutores ensinem aos alunos boas práticas de codificação sem a necessidade de também ensinar programação ou instalar software localmente.

### Principais Funcionalidades

- Edição de documentos com reconhecimento de esquema, incluindo validação, utilizando esquemas acessíveis pela web
- Suporte para folhas de estilo em cascata (CSS) para fornecer uma visualização WYSIWYG dos documentos, bem como uma visualização que exibe as tags
- A capacidade de extrair referências a entidades nomeadas (pessoas, lugares ou organizações) de referências XML já marcadas em um documento, a fim de gerar anotações web
- A capacidade de pesquisar e selecionar identificadores para tags de entidades nomeadas (pessoas, organizações, lugares ou títulos) das seguintes autoridades de Dados Abertos Ligados: [DBPedia](https://dbpedia.org/), [Geonames](https://www.geonames.org/), [Getty](https://www.getty.edu/), [GND](https://www.dnb.de/EN/Professionell/Standardisierung/GND/gnd_node.html), [LINCS Project](https://lincsproject.ca), [VIAF](https://www.viaf.org/) e [Wikidata](https://www.wikidata.org/), ou arquivos de autoridade específicos do projeto.
- Geração de anotações de Dados Ligados correspondentes a entidades nomeadas recém-etiquetadas e anotações de documentos (datas, notas, citações, correções, links, palavras-chave) em XML-RDF ou JSON-LD, em conformidade com o Modelo de Dados de Anotação Web.
- Validação contínua de XML.
- Opções de etiquetagem com restrições de esquema.

### Suporte a TEI

O LEAF-Writer foi projetado para funcionar com [personalizações do esquema da Iniciativa de Codificação de Texto (TEI)](https://tei-c.org/guidelines/customization/#section-1) fornecidas pelo Consórcio TEI. O LEAF-Writer oferece suporte imediato aos seguintes esquemas: [TEI All](https://www.tei-c.org/release/xml/tei/custom/schema/relaxng/tei_all.rng), [TEI LITE](https://www.tei-c.org/release/xml/tei/custom/schema/relaxng/tei_lite.rng), [TEI Simple Print](https://www.tei-c.org/release/xml/tei/custom/schema/relaxng/tei_simplePrint.rng), [jTEI Article](https://www.tei-c.org/release/xml/tei/custom/schema/relaxng/tei_jtei.rng) e [Orlando](https://cwrc.ca/schemas/orlando_entry.rng).

### Esquemas Personalizados

O LEAF-Writer também pode funcionar com esquemas personalizados. Ao abrir um documento, o LEAF-Writer verifica o elemento raiz e a definição do esquema. Atualmente, o LEAF-Writer suporta três elementos raiz diferentes: `TEI`, `ORLANDO` e `CWRCENTRY`. Se o elemento raiz não for suportado pelo esquema, você pode adicioná-lo como um esquema personalizado. O LEAF-Writer salva as informações do esquema no armazenamento local do navegador. Assim, o esquema ficará disponível localmente enquanto você estiver conectado.

## Armazenamento de Documentos

O LEAF-Writer Commons utiliza o GitHub para armazenamento, controle de versão e compartilhamento de documentos. Para aproveitar esses recursos, você precisa estar conectado a uma conta do GitHub. Além disso, você pode abrir documentos colando um XML ou carregando um arquivo do seu computador. Você também pode baixar o arquivo diretamente para o seu dispositivo. Opcionalmente, você pode usar o LEAF-Writer sem uma conta externa. Nesse caso, você só poderá carregar e salvar arquivos no seu computador.

Você pode usar o LEAF-Writer para editar documentos XML ou criar novos documentos a partir de modelos. Há modelos e documentos de exemplo disponíveis aqui para você começar.
