# Local Arch package

This directory contains a local `PKGBUILD` for Arch-based systems such as CachyOS.

Build it from this directory with:

```bash
makepkg -si
```

The package builds for the host architecture:

- `x86_64`
- `aarch64`

This is not an AUR submission recipe. It is just a repo-local Arch packaging
layout you can keep using or adapt later.
